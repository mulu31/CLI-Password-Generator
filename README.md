# 🔐 Simple Password Generator

A lightweight, CLI-based password generator that creates 3 secure passwords based on your preferences. Built with Python using object-oriented programming principles.

## ✨ Features

- **Interactive CLI** - Simple yes/no prompts for easy configuration
- **Multiple Options** - Generates 3 different passwords at once
- **Customizable Character Sets**:
  - Digits (0-9)
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Special symbols (!@#$%^&*()_+-=[]{}|;:,.<>?)
- **Flexible Length** - Generate passwords from 8 to 128 characters
- **Input Validation** - Handles invalid inputs gracefully
- **Loop Mode** - Generate multiple sets without restarting
